Section 3 Important Definitions

3.1 Groups

3.1.1 Group

A group is a set \(\mathsf{G}\) with a binary operation \(\ast\) that satisfies the following:

  1. (Identity) There exists \(e \in \mathsf{G}\) such that \(e \ast g= g \ast e = g\) for all \(g \in \mathsf{G}\).

  2. (Inverse) For all \(g \in \mathsf{G}\), there exists \(g^{-1} \in \mathsf{G}\) such that \(g \ast g^{-1} = g^{-1} \ast g = e\).

  3. (Associativity) \((ab)c = a (bc)\) for all \(a,b,c \in \mathsf{G}\)
